# Enterprise Network Design using Cisco Packet Tracer

## 📌 Overview

This project is an academic network design implementation created using Cisco Packet Tracer as part of my Computer Engineering studies.

The network was designed for a company consisting of multiple departments with different host requirements. VLSM subnetting was used to efficiently allocate IP addresses and improve network organization.

---

## 🛠️ Technologies & Concepts

* Cisco Packet Tracer
* VLSM Subnetting
* Router Configuration
* Network Addressing
* Connectivity Testing
* Basic Network Security

---

## 🏢 Departments

The network contains the following departments:

* Marketing
* Sales
* IT
* Finance

Each department was assigned to a separate subnet.
